CSS Flexbox

display: flex | inline-flex Used to designate a div as a flex container, places elements inline with eachother

justify-content: flex-start | flex-end | center | space-around | space-between To position items from left to right

align-items: flex-start | flex-end | center | baseline | stretch Makes it possible to space flex items vertically

flex-grow: 1 | 2 | 3 Allows us to specify if items should grow to fill a container

flex-shrink: 1 | 2 | 3 Used to specify which elements will shrink and in what proportion

flex-basis: 100px; Allows us to specify the width of an item before it stretches or shrinks

flex: 2 1 150px; Gives us a shorthand property to specify how much items stretch and shrink (can declare flex-grow, shrink, basis all in 1 link)

flex-wrap: wrap | nowrap | wrap-reverse Allows items to move to the next line when necessary

align-content: flex-start | flex-end | center | space-between | space-around | stretch Used for a flex container that has multiple rows

flex-direction: row | row-reverse | column | column-reverse Let's us change the axis, they are interchangeable between horizontal and vertical

flex-flow: column wrap; Let's us define both flex-wrap and flex-direction in 1 line

CSS Grid

display: grid; Turns the container into a grid

grid-template-columns: 100px 50% 200px; Specifies the widths of each column, each number counts as a column

grid-template-rows: 10% 20% 600px; Specifies the widths of each row

grid-template: 200px 300px / 20% 10% 70%; Shorthand of both of the above, values after the slash determine the size of each column

grid-template: 2fr 1fr 1fr / 1fr 3fr 1fr; Fractions make it easier to not overflow the grid, rows = height, column = width

grid-template: repeat(3, 1fr) / 3fr 50% 1fr; The repeat() function will repeat 3 times for 1fr per row or column

grid-template: 100px minmax(100px, 500px) 100px; The minmax allows columns or rows getting too small or big

row/column-gap: 10px; Will put some gap between each row and column (gap: 20px 5px shorthand)

grid-row-start: 1; Edits the item to start on a specific row

grid-row-end: 7; Edits the items inside the grid container to span over multiple rows, should be one greater than the last row you want to cover

grid-row: 4 / 6; Shorthand for start and end

grid-column: 4 / span 2; Has the same abilities as above

grid-area: 2 / 3 / 4 / span 5; Takes 4 values in this order grid-row-start / column-start / row-end / column-end;

grid-template-areas: "header header" "nav nav" "left right" "footer footer"; Written on a different line,

grid-area: header | nav | left | right | footer Use this property to assign what each area is with template areas

justify-items: start | end | center | stretch Positions grid items along the inline (row) axis

justify-content: start | end | center | stretch | space-around | space-between | space-evenly Positions grid items along the inline-block (row) axis

align-items: start | end | center | stretch Positions grid items along the block (column) axis, top to bottom

align-content: start | end | center | stretch | space-around | space-between | space-evenly Positions the rows along the block (column) axis, top to bottom, declared on grid containers

justify-self: start | end | center | stretch Positions an individual item in the grid along the row axis

align-self: start | end | center | stretch Positions an individual item in the grid along the column axis

implicit grid vs explicit grid start | end | center | stretch Positions an individual item in the grid along the column axis

grid-auto-rows: px | % | fr | repeat(); Specifies the height of implicitly added grid rows

grid-auto-columns: px | % | fr | repeat(); Specifies the width of implicitly added grid columns, defined on grid containers

grid-auto-flow: row | column | dense or row/column dense; Specifies whether new items should be added to rows or columns, declared on containers.
